OCR 识别设置

配置步骤：
1. 注册百度智能云（前往 →），创建「文字识别」应用，获取 API Key 和 Secret Key
2. 注册 Cloudflare（前往 →），在 Workers & Pages 创建 Worker
3. 将 ocr-worker.js 代码粘贴进去，在 Settings → Variables 中设置：
    BAIDU_API_KEY = 你的 API Key
    BAIDU_SECRET_KEY = 你的 Secret Key
4. 部署后复制 Worker URL，粘贴到上方输入框
